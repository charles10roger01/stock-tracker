# 強勢股追蹤儀表板 - 設定說明

## 第一步：建立 GitHub Repository

1. 登入 GitHub（github.com）
2. 右上角點「+」→「New repository」
3. Repository name 填：`stock-tracker`
4. 選「Public」
5. 勾選「Add a README file」
6. 點「Create repository」

---

## 第二步：上傳檔案到 GitHub

1. 進入剛建立的 repository
2. 點「Add file」→「Upload files」
3. 把以下四個檔案拖進去上傳：
   - `index.html`
   - `stocks.json`
   - `update_prices.py`
   - `每日更新收盤價.bat`
4. 點「Commit changes」

---

## 第三步：開啟 GitHub Pages

1. 進入 repository，點上方「Settings」
2. 左側選「Pages」
3. Source 選「Deploy from a branch」
4. Branch 選「main」，資料夾選「/ (root)」
5. 點「Save」
6. 等約 2 分鐘，網址會出現：
   **https://charles10roger01.github.io/stock-tracker/**

---

## 第四步：取得 GitHub Token

1. GitHub 右上角頭像 →「Settings」
2. 左側最下方「Developer settings」
3. 「Personal access tokens」→「Tokens (classic)」
4. 「Generate new token (classic)」
5. Note 填：`stock-tracker`
6. Expiration 選「No expiration」
7. 勾選「repo」（第一個大項目）
8. 點「Generate token」
9. **複製這串 token，只會顯示一次！**

---

## 第五步：設定 Python 腳本

1. 用記事本開啟 `update_prices.py`
2. 找到這行：
   ```
   GITHUB_TOKEN = ""
   ```
3. 在引號裡面貼上你的 token，例如：
   ```
   GITHUB_TOKEN = "ghp_xxxxxxxxxxxxxx"
   ```
4. 儲存檔案

---

## 每天使用方式

### 新增股票：
1. 打開儀表板網址
2. 點「+ 新增股票」
3. 輸入代號、名稱、族群
4. 點「匯出清單」下載 stocks.json
5. 到 GitHub repository 更新這個檔案

### 每日更新收盤價：
1. 每天下午 2 點後（收盤後）
2. 雙擊「每日更新收盤價.bat」
3. 等待完成，自動上傳到 GitHub
4. 打開儀表板即可看到最新資料

---

## 常見問題

**Q：腳本說「無法取得」某檔股票？**
A：可能是上櫃股票，或當天沒有成交。隔天再試試看。

**Q：網頁顯示「找不到 stocks.json」？**
A：請確認 stocks.json 已上傳到 GitHub repository，且 GitHub Pages 已啟用。

**Q：GitHub Token 過期了怎麼辦？**
A：重新到 GitHub 產生新的 token，更新到 update_prices.py 裡。
